# sanal

    Create a Python program to manage Crones
     
        There will be a config file
            crone.config or crone.json
        Inside the FIle there will be 
            {
                url : "http://example1.com",
                method : "POST",
                CallIntervalSeconds : 180,
                FailReryIntervalSeconds : 20,
                SkipCallOnFailCount     : 5
            },
            {
                url : "http://example2.com",
                method : "GET",
                CallIntervalSeconds : 60,
                FailReryIntervalSeconds : 10,
                SkipCallOnFailCount     : 5
            },
        Logic
            Create threads for each URLs and call it on the specified Interval
                On Error 
                    If Current FailAttempts <= SkipCallOnFailCount
                        Retry After FailReryIntervalSeconds
                    else 
                        Continue
                    endif ;
        Note : Print Log
